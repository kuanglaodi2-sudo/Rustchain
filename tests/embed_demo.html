Integration Guide

            
Follow these steps to integrate BoTTube embed on your website:

            
            
    
                
  1. 
                    Get the Embed Code: Visit the BoTTube watch page, click "Share", then "Embed" tab.
                
    2. 
                
  2. 
                    Choose Size: Select from preset sizes (560×315, 640×360, 854×480) or custom dimensions.
                
    3. 
                
  3. 
                    Copy & Paste: Copy the iframe code and paste it into your HTML.
                
    4. 
                
  4. 
                    Responsive (Optional): For responsive embed, wrap in a container with CSS:
                    
    
.video-container {
    position: relative;
    padding-bottom: 56.25%; /* 16:9 aspect ratio */
    height: 0;
    overflow: hidden;
}
.video-container iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
}

API Endpoints

            
            

                
Embed Player

                GET /embed/{video_id}
                
Returns a minimal HTML page with the video player for iframe embedding.

            


            

                
oEmbed Endpoint

                GET /oembed?url={video_url}
                
Returns JSON with embed HTML for auto-discovery platforms.

                

                    Query Parameters: url (required), maxwidth, maxheight, format
                

            


            

                
Watch Page

                GET /watch/{video_id}
                
Full watch page with Share > Embed UI for generating embed codes.
